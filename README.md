# Hi 👋 My name is Uğur Dalkıran

## Frontend Developer

I started my software adventure by publishing my personal website in December 2011. Throughout this adventure, I continue on my way by developing many websites and mobile applications as a freelance. I publish some of the works I have developed on my personal website (ugurdalkiran.com). I enjoy writing frontend mainly JavaScript and CSS. Recently, I have been working mainly on ReactJS and React Native. Developing mobile applications and coding with React Native motivates me a lot.
